# Enterprise-Help-Desk-Ticketing-Lab-
A techincal lab displaying a corporate IT support environment . This poroject consists of a full-stack ticketing system on a virtualized Windows Server managing end the end-to-end incident lifecycle. 
Technologies Used

OS : Windows Server 2025 (Standard Evaluation)
Web Server: Internent Information Services (IIS)
Database MySQL 
Application: osTicket (PHP-based)
Virtualization: UTM/QEMU
Environment: FastCGI, PHP 8.d

Server Administration: Installing an configuring IIS and managing NTFS permissions for web directories
Database Managment: Schema intialization and connectivity troubleshooting between MySQL and PHP
Access Control: Implementing Role-Based Access Control (RBAC)
Resolving configuration issues with FastCGI handlers and system dependecies
